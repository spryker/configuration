# Configuration Module
[![Latest Stable Version](https://poser.pugx.org/spryker/configuration/v/stable.svg)](https://packagist.org/packages/spryker/configuration)
[![Minimum PHP Version](https://img.shields.io/badge/php-%3E%3D%208.3-8892BF.svg)](https://php.net/)

Configuration module provides ability to define module configuration in a declarative way and manage it in Backoffice.

## Installation

```
composer require spryker/configuration
```
